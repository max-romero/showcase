# FAQ

## 1. Why should I use it if I already have Google Maps?
Google Maps is excellent for finding *where* things are, but it doesn't tell you *what* it's like to live there.

## 2. Why is not completely free?
The solution uses AI-generative narrative that requires token consumption that is not free.

## 3. Will new parameters be added? (e.g., homicides, supermarkets, nightlife)
Yes. The plan is to progressively upload new categories for analysis.
