# solution overview

Data‑driven web application that provides an instant, AI‑generated quality‑of‑life report for any address or point of interest in the Autonomous City of Buenos Aires (CABA).  
It helps residents, real‑estate agents, and investors understand a neighborhood’s strengths and weaknesses before moving, buying, or renting.

The result is a frictionless experience: the user types an address, and within seconds they see a human‑readable summary of what the area offers and where it falls short.

# AI application

The project applies artificial intelligence at two key points: **spatial intelligence** (PostGIS + configurable rules) and **generative language** (LLM). 
The combination of both makes the output far more valuable than a simple list of numbers.
**Generative AI narrative** – Feeds the structured counts and examples into a LLM to produce a fluent, honest, Spanish‑language pros/cons analysis with a practical recommendation.
